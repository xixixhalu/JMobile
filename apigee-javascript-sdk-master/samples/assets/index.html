Apigee Entity Sample App

This sample app will show you how to perform basic operations on assets using the Apigee JavaScript SDK, including:

  • creating an asset
  • Adding binary data to an asset
  • Downloading asset content

IMPORTANT - BEFORE YOU BEGIN

Be sure the Apigee JavaScript SDK (apigee.js) is properly included in this project.

See /samples/Readme.md for more information on including the SDK.

Apigee Entity Sample App

This app show the basics of working with assets using the Apigee JavaScript SDK. Click each button to execute an API call and see its response. For this app to work properly you must run the create step first.

To view the code and more detailed comments on how this app operates, open js/index.js.

    This creates an asset that is owned by our new user with the name and path specified.

  3. Add content
